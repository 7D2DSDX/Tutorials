SDX Intermediate Tutorial




The SDX Intermediate Tutorial assumes you have already followed the "Getting Set up" guide and completed the "SDX Beginner Tutorial"



In our Beginner Tutorial examples, we added in the Katana and a Cube block to the game. These mods added a new unity3d object, and XML changes. What it didn't do, was add any new custom scripts.


There's two types of scripts that can be added in by SDX:  PatchScripts and Scripts.


Patch Scripts are scripts that change the Assembly-CSharp.dll at compile time.


Scripts are scripts that are compiled into the Mods.dll, and loaded at run time.


For the Intermediate Tutorial, we'll explore the Patch Scripts.



HAL9000 has created a series of extremely valuable SDX 7 Videos that take you step by step through some of the advanced methods


The 7D2D SDX 7 Patch Scripts video takes you step by step through in how to create your first PatchScript.
