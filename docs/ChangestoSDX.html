Changes to SDX

Getting Started ›› Getting the Tools ›› SDX 0.7.0 ››

Note: If you downloaded using the recommended SDXModding Download link, you should be aware that this is a modified SDX version by sphereii.



The following Quality of Life Changes were made. These changes were done to make the experience more comfortable for new modders, and help players install files easier.



- Skipping Localization.txt due to encoding issues

- Skipping XUI files, due to encoding issues

       - The XUI files contain variables, such as ${name}, and the XML process is filtering them out.

- Removed the requirement to having a Settings.ini file

- Changed the Mods folder structure for clients to more closely match traditional XML Mods folders

- Contains the empty Mods.dll, and have it automatically deploy as part of the SDX Compile process

- Copies the Config.xml into the correct location



Example of the revised Folder Structure:


Any Resources you use, such as messes, should be located under Mods\<ModName>\Resources.


Example:


7daystodie.exe

Config.xml

Mods\HalDllUpdates

Mods\True Survival SDX\Resources
