About 7D2D SDX Project

The 7D2D SDX Project was created to help those who are struggling with working with SDX, and those who were too unsure of how to even get started.


We came together and joined to create a comprehensive tutorial system to help guide you through your SDX journey, running from simple SDX instrumentation up to advanced topics, like C# Scripting and creating completely new items in Unity.


We leveraged the following Tools to make this tutorial a success:


- GitHub.com: Free hosting, and free web service are a huge boon to this project. However, the key to this site is the ability to fork the project, and allow other people to continue the work.


- HelpNDoc: The software was used to create this tutorial system.


- Visual Studio Community 2017: A development environment to help us build our mods
