Here's some examples of what people have been doing with SDX recently!

Display Cases ( Unreleased ):

Halicopter:


This mod adds in some new craftable items that spawn in a Halicopter™ to come pick the item up and return it to base. 


https://www.youtube.com/watch?v=bzSDvJmDlJk




Zombies Run In the Shadows:


Quick mod to re-enable zombies running in dark areas. It's not exactly as it used to be in vanilla. Torches won't slow them down and shade from buildings or the environment don't effect them but it adds another level of danger to going into those deep, dark places.


https://www.youtube.com/watch?v=_KStkdkAfVI&feature=youtu.be




three08


Hidden Iron Bars:


Adds 2 powered iron bars that sit in the ground or roof and pop up or down when powered. This still has some issues I'm working on, so make sure you read the issues list before installing.


https://www.youtube.com/watch?v=f6-XhHexKd4



Aquarium Mod:


Aquarium with fish swimming in it. Lights up when activated. (No power requirements for now) The light is traveling too far and is too bright, I'll try to get that sorted out soon.


https://www.youtube.com/watch?v=zp0aee2IGv4


Beacon


A beacon that lights up and rotates when powered. It looks for power in lots of spots, 2 blocks to the left or right, 2 blocks directly above or bellow and 2 blocks to the front or back. If any of these location contain a powered relay the beacon will be powered. I'll see if I can find some electric motor sound for it too.


https://www.youtube.com/watch?v=o5HGuFi-Dwk
